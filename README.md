# Сервис голосования


## Backend

Запуск в режиме разработки: `npm run start:dev`

Сборка: `npm run build`


## Frontend

Запуск в режиме разработки: `npm start`

Сборка: `npm run build`


## Website JS SDK

Автосборка бандла в режиме разработки: `npm start`

Тестирование realtime: `npm run test`

Сборка: `npm run build`
